# Öğrenci Not Ortalaması Hesaplayıcı

Bu proje Python öğrenme sürecimde geliştirdiğim basit bir konsol uygulamasıdır.

## Proje Amacı

Kullanıcıdan alınan 3 notun ortalamasını hesaplayarak harf notu ve geçme durumunu gösterir.

## Kullanılan Teknolojiler

- Python

## Özellikler

- Ortalama hesaplama
- Harf notu belirleme
- Geçti / Kaldı sonucu gösterme
- Hatalı giriş kontrolü
